# Store Finder App

A simple Flask app to list stores and show their details.

## How to Run

1. Clone the repo
2. Run `pip install flask`
3. Run the app with `python app.py`
4. Go to `http://127.0.0.1:5000`

## Features

- Homepage showing all stores
- Store detail page
